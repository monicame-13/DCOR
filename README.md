# DCOR
Code and example data set showing the data evaluation to extract concentration dependent diffusion coefficients from optical data via Boltzmann-Matano analysis. As described in M. Mead, N. Kerner, R. Talei, Y. Joshi, G. Schmitz, Lithium transport from operando microscopy: Kinetic trapping in lithium manganese oxide, Nano Energy 151 (2026) 111797. https://doi.org/10.1016/j.nanoen.2026.111797

To run the evaluation code (DCOR example code.ipynb) you have to additionally download the raw data from [Zenodo](https://doi.org/10.5281/zenodo.19113446) and save it in the same folder as the jupyter notebook.

## Getting Started
1. Clone or download this repository
2. Download the raw data from Zenodo: [https://doi.org/10.5281/zenodo.19113446]
3. Place the data files in the same folder as the notebook
4. Install dependencies: pip install -r requirements.txt
5. Open the notebook: jupyter notebook "DCOR example code.ipynb"
